Interpol - MIF31 - projet

Interpol

Introduction

Voici notre projet de programmation générique MIF31.
Le document suivant va énoncer assez rapidement tout ce qu'il faut savoir.

Conditions d'utilisations

Il est nécessaire de respecter les conditions suivantes pour essayer notre projet :
  • Avoir GraPic
  • Avoir SDL2, SDL2_image, SDL2_ttf

Lancement

Pour essayer notre projet, il faut savoir qu'il existe une partie graphique.
C'est la partie par defaut, et qui se contente d'un espace en 2 dimensions.
Il suffit juste de :
make exec
Ou bien
./exec/myprog

Il est également possible de générer des points aléatoirements via le script generate.py présent dans ce répertoire, de la façon suivante :
./generate.py Dimension_des_points nombre_de_points_voulu

Interactions

Une fois l'interface graphique chargée, en plaçant votre souris dans un simplex, vous aurez une prévisualisation du point d'interpolation qu'elle représente.

Puis, un clic gauche permet de fixer ce point, avec la valeur d'interpolation calculée.
Et ainsi de suite...

Informations complémentaires

  • Il est possible de lancer des tests unitaires comme ceci :
    make test
    Cependant, une version de perl doit etre installée.
  • le fichier valgrind.txt est un fichier de suppression pour l'application, ignorant tous les problèmes liés à GraPic (donc à SDL2) lors de l'execution de valgrind. Pour l'utiliser, il faut faire :
    valgrind --leak-check=full --num-callers=50 --show-reachable=yes --suppressions=valgrind.txt ./exec/myprog
  • Il est possible de compiler la documentation en faisant :
    doxygen doc/doxyfile
  • Le répertoire bin/ contient les DLLs requises pour Grapic
